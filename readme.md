# Grym & Grim - Rock Band Landing Page

A modern and responsive landing page for the fictional Ukrainian rock band "Grym & Grim" (Thunder & Roar). This project is built with modern web technologies and demonstrates a clean, component-based approach to frontend development.

- **Fully Responsive Design**: Adapts seamlessly to all screen sizes, from mobile phones to desktops.
- **Interactive UI**: Features include a mobile burger menu, a modal window for ticket orders, and hover effects.
- **Smooth Scrolling**: Smooth navigation between page sections using anchor links.
- **Client-Side Form Validation**: JavaScript-powered validation for the contact and ticket order forms.
- **Adaptive Images**: Uses the `<picture>` tag to serve optimized images for different screen resolutions.
- **Component-Based SCSS**: Styles are organized into logical blocks using a BEM-like methodology.
